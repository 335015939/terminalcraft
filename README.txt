# terminalcraft

game for terminal

a thing i thought of, somewhat inspired on terraria and minecraft

./comp to compile and ./run to run

controls:
    awsd:move around
    AWSD:change where your facing(without moving)
    j:use current item
    p or esc:pause
    q:quit
    i:inventory
    c:craft
    k:interact with block
    / or \:drop item
    l:pick up item
    e:open equip menu
    t:open player stats


tested and compiled and coded with Gitpod
